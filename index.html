HTML5 memungkinkan untuk menampilkan berbagai konten multi media

  1. <image>

CSS

position

CSS3 memberikan berbagai macam metode penempatan seperti antara lain:

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

float

Float merupakan sebuah properti yang memungkinkan kita memposisikan suatu objek terhadap suatu parent perlu diperhatikan menggunakan float menyebabkan tidak normal Document Flow

  1. Right
  2. Left

display

Display merupakan properti yang mengatur bagaimana sebuah element ditampilkan

  1. Inline
  2. Block
  3. Table
  4. list-item
